`dataset` 目录用于存放数据集

## 目录结构

参照 VOC 数据集

```
|____dataset
  |
  |____Annotations
  |
  |____ImageSets
  | |
  | |____Main
  |
  |____JPEGImages
```

说明：

Annotations 存放 xml 的标注文件

ImageSets/Main 存放 txt 的图片说明

JPEGImages 存放所有的图片
